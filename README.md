## 介绍
微信小程序：用于获取用户30天内步数，并可将所有用户步数导出到表格中进行统计分析。
主要涉及技术包括 JavaScript、PHP、Python。

微信端通过后台的解密js来获取步数，并将步数提交给服务器，服务器使用PHP将步数导出为csv文件，最后管理员在本地电脑使用Python将数据导出至Excel表格。
